# 🛸 NEON GLIDE

매일 바뀌는 네온 코스를 활공하는 원버튼 하이퍼캐주얼 웹게임.

**▶ 플레이: https://csj3814-create.github.io/neon-glide/**

## 조작
- 누르고 있으면 상승, 떼면 하강 (마우스/터치/스페이스바)
- 네온 게이트 틈을 통과하면 +1점, ⭐ 별을 먹어도 +1점
- 움직이는 게이트와 🔴 점멸 레이저를 조심하세요 — 레이저는 꺼진 순간에 통과!

## 데일리 챌린지
날짜(KST) 기반 시드로 코스가 생성되어, 같은 날에는 전 세계 모든 플레이어가 동일한 코스에서 경쟁합니다. 점수 공유 버튼으로 친구에게 도전장을 보내세요.

## 기술
- 의존성 없는 단일 `index.html` (Canvas 2D + WebAudio 신스웨이브 BGM)
- 저비용 레이어드 글로우 렌더링 (shadowBlur 미사용)
